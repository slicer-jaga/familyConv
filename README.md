# familyConv
Программа для конвертации баз приложения [Family 12](http://www.sanuel.com/ru/family/) в формат CSV.
* Делалось под формат [Дзен-мани](https://zenmoney.ru/), но подойдет и для некоторых других сервисов и приложений.
* После выгрузки рекомендуется сверить остатки по счетам в приложении, и в файле `<Название_базы_данных_bill.csv>`. Если данные совпадают, то, скорее всего, выгрузка была успешна.
 
## Формат выгружаемого файла
`Дата;Категории и теги;Контрагент;Счет списания;Сумма списания;Счет зачисления;Сумма зачисления;Комментарий`

## Использование
`$ ./familyConv "<Файл_базы_данных>.fdb"`
`$ ./familyConv /?`
