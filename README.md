# familyConv
Программа для конвертации баз приложения [Family 12](http://www.sanuel.com/ru/family/) в формат CSV.
* Делалось под формат [Дзен-мани](https://zenmoney.ru/), но подойдет и для некоторых других сервисов и приложений.
* После выгрузки рекомендуется сверить остатки по счетам в приложении, и в файле `<Название_базы_данных_bill.csv>`. Если данные совпадают, то, скорее всего, выгрузка была успешна.

## Изменения
### 0.1.1
* [+] [BudgetBakers](https://budgetbakers.com/)
 
## Формат выгружаемых файлов
### Дзен-мани
Файл со всеми транзакциями и файл со счетами.
`Дата;Категория;Получатель;Счёт;Сумма (расход)";Счёт-получатель перевода;Сумма (доход);Комментарий;Теги`

### BudgetBakers
Отдельные файлы для каждого счета.
`Дата;Категория;Получатель;Сумма;Заметка;Метки`

## Использование
```
$ ./familyConv "<Файл_базы_данных>.fdb"
```
```
$ ./familyConv "<Файл_базы_данных>.fdb" /c "<Тип выгрузки>"

Тип выгрузки:
		zm - Дзен-мани
		bb - BudgetBakers
```
```
$ ./familyConv /?
```
